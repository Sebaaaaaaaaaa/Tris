# SUPER TRIS

Super Tris is a java based game created to improve my java programming skills and do sum new and cool, like you know what i mean.

## Features
- PvP mode
- PvE mode
	- Easy
	- Medium 
	- Impossible
- Input validation
- Simple board display

## Installation

Clone the repository:

git clone https://github.com/Sebaaaaaaaaaa/Tris.git

Enter the project directory:

cd Tris

Compile the program:

javac *.java

## Usage

Run it:

java tris.Main
